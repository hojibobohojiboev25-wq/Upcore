# Success Tracker Pro (React Native + JavaScript)

Текущая версия: `8.0.0`

Профессиональный мобильный трекер успеха:

- управление задачами (приоритет, заметки, фильтры);
- управление целями (числовой прогресс, быстрые шаги `+1/-1`);
- аналитика продуктивности (KPI, прогресс-метрики, достижения);
- серия дней (streak) и история за последние 7 дней;
- локальное сохранение данных на устройстве (`AsyncStorage`).
- v8-улучшения: продуктивность KPI, дневная цель задач, поиск и сортировка задач.

## Запуск

1. Установить зависимости:

```bash
npm install
```

2. Запустить Expo в LAN-режиме (по умолчанию):

```bash
npm run start
```

3. Открыть приложение в Expo Go на телефоне или в эмуляторе.

Если нужно очистить кэш Metro:

```bash
npm run start:clear
```

Если LAN недоступен, попробуй tunnel:

```bash
npm run start:tunnel
```

## Структура

- `App.js` - корневая навигация (Bottom Tabs)
- `src/context/SuccessContext.js` - состояние приложения + бизнес-логика
- `src/screens/` - экраны: Главная, Задачи, Цели, Аналитика
- `src/components/` - переиспользуемые UI-компоненты
- `src/storage/storage.js` - сохранение/загрузка состояния

## Важно для Expo Go

- телефон и ПК должны быть в одной Wi-Fi сети;
- если видишь `exp://127.0.0.1:...`, это localhost и телефон не подключится;
- в таком случае перезапусти: `npm run start` и используй URL вида `exp://192.168.x.x:8081`;
- при проблемах выключи VPN/прокси и запусти `npm run start:clear`.

## Сборка iOS/Android (EAS)

1. Логин в Expo аккаунт:

```bash
npx eas login
```

2. Сборка Android (AAB/APK):

```bash
npm run build:android
```

3. Сборка iOS (IPA):

```bash
npm run build:ios
```

4. Сборка обеих платформ:

```bash
npm run build:all
```
